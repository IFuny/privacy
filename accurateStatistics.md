# 如何做到精确统计

1. 点开本app内的`设置`页面
2. 打开设置页面里的`及时统计`开关
3. 在弹出的定位权限弹窗选择`使用App期间`
4. 滑到底部，点击`跳到->设置`,然后会自动跳转到系统的设置页面
5. 在系统设置页面选择`位置`-->`始终`
